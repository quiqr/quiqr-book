                    Quiqr Forms Quickstart #

                    To quickly understand how Quiqr Forms works, try to implement the example form configuration below in your Quiqr website. After implementing the form, you need to publish your website to the Quiqr Cloud.

                    Example form configuration #

                    Below is a minimal form html to use in your Hugo Layout. You need to put QUIQR_FORM in the form action to use the Quiqr Forms service.

                    <form method="post" action="QUIQR_FORM">
  Name: <input type="text" name="name" /><br/>
  Email: <input type="text" name="email" /><br/>
  <textarea name="message" placeholder="Message"></textarea><br/>
  <input type="submit" value="send" />
  <input type="hidden" name="QUIQR_FORM_ID" value="default">
</form>

                    Here’s a configuration belonging to above form in the file quiqr/forms/default.yml

                    ---
subject: Website contact form
to: info@example.com
redirect_to: /contact/thanks/
senders_email_field: email
reply_to_sender: true
antispam: slide
validations:
  fields:
    email:
      type: email
      empty: false
      mandatory: true
    name:
      type: string
      empty: false
      mandatory: true
    message:
      type: string
      empty: false
      mandatory: true
