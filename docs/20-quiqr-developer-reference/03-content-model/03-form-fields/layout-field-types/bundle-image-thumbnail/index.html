                    Bundle image thumbnail #

                    The bundle-image-thumbnail field renders a thumbnail image. It can only be used together with the bundle-manager field. Read [bundle-manager documentation](https://book.quiqr.org/docs/20-quiqr-developer-reference/03-content-model/03-form-fields/container-field-types/bundle-manager/) for more information.

                    Properties #

                    propertyvalue typeoptionaldescription
                    keystringmandatoryKeys are for internal use and must be unique

                    Sample #

                    Configuration #

                    ./quiqr/model/base.
                         
                    key: thumb
type: bundle-image-thumbnail

                    key = "thumb"
type = "bundle-image-thumbnail"

                    {
   "key": "thumb",
   "type": "bundle-image-thumbnail"
}
